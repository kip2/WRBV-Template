# WASM-Template-with-Rust-Vite-Bacon

Project template.

WASM + Rust + Bacon + Vite

### Intall

Clone the repository and install dependencies.

```sh
git clone https://github.com/kip2/WRBV-Template
cd WRBV-Template
npm install
```

### Start Develop

```sh
npm run start
```
